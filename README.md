# Employee Salary Analysis

## Overview

This project analyzes employee salary data using Python, Pandas, and Matplotlib.

## Features

* Data Cleaning
* Duplicate Removal
* Salary Statistics
* GroupBy Analysis
* Pivot Tables
* Data Visualization
* CSV Export

## Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Output

* Department-wise salary analysis
* Pivot table reports
* Salary distribution chart
* Cleaned CSV file
